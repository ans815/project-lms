# 📚 Library Management System (Flask + MySQL + Docker)

## 🚀 Description
This is a Library Management System built using Flask. It allows users to manage students, authors, and books from a single dashboard.

## 🔥 Features
- Student Registration & Login
- Author Registration & Login
- View All Students
- View All Authors
- Dashboard UI
- MySQL Database Integration
- Dockerized Deployment
- Hosted on AWS EC2

## 🛠️ Tech Stack
- Python (Flask)
- MySQL
- Docker
- AWS EC2
- HTML, CSS
